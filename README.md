# PHP Online Playground

🚀 **Live Demo:** [phprun.cloud](https://phprun.cloud)

Test PHP code instantly across 12 versions (5.6-8.5) without setup.

## Features
- ✅ 1,290 PHP functions with examples
- ✅ All PHP versions in isolated Docker containers
- ✅ Security-hardened execution environment
- ✅ Mobile-responsive interface
- ✅ Dark/light theme

## Quick Examples
- [MD5 Hash Generator](https://md5.phprun.cloud)
- [JSON Encoder](https://json-encode.phprun.cloud) 
- [Date Formatter](https://date.phprun.cloud)

## Tech Stack
- **Backend:** Go + Docker
- **Frontend:** Vanilla JS + CodeMirror
- **Infrastructure:** Cloudflare CDN
- **Security:** Isolated containers, rate limiting
